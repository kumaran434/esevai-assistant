# மொழித் தெரிவு (Language Preference)

- பயனர் ஆங்கிலத்தில் கேட்டாலும், பதில்கள் அனைத்தும் **தூய தமிழில்** மட்டுமே இருக்க வேண்டும்.
- தங்கிலீஷ் (Tanglish) தவிர்க்கப்பட வேண்டும்; தமிழ் எழுத்துக்களை மட்டுமே பயன்படுத்த வேண்டும்.

# பிழை திருத்த வரலாறு மற்றும் கட்டுப்பாடுகள் (Fixed Issues & Constraints)

இங்கே குறிப்பிடப்பட்டுள்ள பிழைகள் ஒருமுறை சரிசெய்யப்பட்டுள்ளன. இனி வரும் எந்த மாற்றங்களிலும் இந்தப் பழைய பிழைகள் மீண்டும் வரக்கூடாது:

1. **Electron Main Logic**: பிரதான செயல்பாடுகள் `electron-main.cjs` கோப்பில் இருக்கும். ஆனால் தர்க்கங்கள் (Logic) `main-process/` ஃபோல்டரில் பிரிக்கப்பட்டுள்ளன:
   - `portalWin`, `assistantView`, `lastFile` ஆகிய மாறிகள் இப்போது `main-process/ipc-handlers.js` கோப்பின் மேல் பகுதியில் அறிவிக்கப்பட்டுள்ளன.
   - ஆட்டோமேஷன் தர்க்கங்கள் `main-process/automation-logic.js` கோப்பில் உள்ளன.
2. **Path Resolution**: Production-இல் `dist/index.html` கோப்பைக் கண்டறியும் பாதை எப்போதும் சரியாக இருக்க வேண்டும் (`path.join(__dirname, 'dist', 'index.html')`).
3. **Build Cleanup**: மாற்றங்களைச் செய்த பிறகு `npm run clean` மூலம் பழைய 'dist' தரவுகளை நீக்கிய பிறகு புதிய 'build' செய்யப்பட வேண்டும். `rimraf` பயன்படுத்தப்படுகிறது.
4. **Preload Script**: Automation preload script (`preload-automation.cjs`) எப்போதும் `app.getPath('userData')` பகுதியில் சேமிக்கப்பட வேண்டும். இது `main-process/automation-logic.js` மூலம் கையாளப்படுகிறது.

**முக்கிய விதி**: ஒவ்வொரு முறையும் நீங்கள் குறியீட்டை மாற்றும் முன், இந்த `AGENTS.md` விதிகளைப் படித்து, தற்போது செய்யும் மாற்றம் இந்தப் பழையFix-களை பாதிக்கவில்லை என்பதை உறுதி செய்யவும்.

# புதிய முன்னுரிமை விதி (Priority Rule)

- நீங்கள் குறிப்பாக "Web App" என்று சொல்லாதவரை, நான் செய்யும் அனைத்து மாற்றங்களும் **டெஸ்க்டாப் ஆப் (Desktop App)**-க்காக மட்டுமே இருக்க வேண்டும்.
- வெப் ஆப் (Web App)-இல் போர்ட்டல்களைக் கிளிக் செய்யும்போது புதிய டேப் (New Tab) திறப்பதைத் தடுத்து, அதற்குப் பதிலாக "டெஸ்க்டாப் ஆப் பதிவிறக்க" அறிவுறுத்தலை மட்டுமே காட்ட வேண்டும்.
- நீங்கள் **"ஆப்பில் எதுவுமே பண்ணாதே"** என்று சொன்னால், நான் குறியீட்டில் (Codebase) எந்த ஒரு மாற்றத்தையும் செய்யக்கூடாது. இது ஒரு மிக முக்கியமான விதி.
